Alerts

Fornecer mensagens de feedback contextual para ações típicas do usuário com vários tipos de mensagens de alerta disponíveis e flexíveis.

Exemplos básico de uso

Os alertas estão disponíveis para qualquer comprimento de texto, bem como um botão de demissão opcional. Para um estilo adequado, use uma das oito classes contextuais necessárias (por exemplo, .alert-success).

Fundo

Além dos fundos já pré definidos, você pode usar os fundos default, 3d e flat. O Default é o fundo padrão, para utilizar as outras opções basta usar as classes bkg-flat-[nome-da-cor] e bkg-3d-[nome-da-cor].
Por exemplo: .bkg-3d-success e .bkg-flat-success
