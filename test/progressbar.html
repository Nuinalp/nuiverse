Barra de progresso

Documentação e exemplos para o uso de barras de progresso personalizadas do NUIverse com suporte para barras empilhadas, origens animadas e rótulos de texto.

Como funciona

Os componentes de progresso são criados com dois elementos HTML, alguns CSS para definir a largura e alguns atributos. Nós não usamos o elemento <progress> HTML5, garantindo que você pode empilhar barras de progresso, animá-las e colocar etiquetas de texto sobre elas.

  • Utilizamos o .progress como um invólucro para definir o valor máximo da barra de progresso.
  • Usamos a barra de progresso interna para indicar o progresso.
  • O .progress-bar requer um estilo inline, classe de utilitário ou CSS personalizado para definir sua largura.
  • O .progress-bar também requer alguns atributos de função e aria para torná-lo acessível.

Estilo padrão básico

O NUIverse oferece vários recursos para ajudar você a definir a largura e altura de seus objetos. Dependendo de suas necessidades, isso pode ajudar a configurar rapidamente o progresso.
O exemplo abaixo utiliza a classe w-75, nossas classes de largura e altura podem ser usadas em qualquer elemento, basta que você insira a classe .w-[tamanho] ou h-[altura]. A largura e altura nessas classes são medidas em porcentagem váriando de 5 em 5 até 100%.

Altura

Nós apenas estabelecemos um valor de altura no .progress, então, se você alterar esse valor, a barra de progresso interna redimensionará automaticamente em conformidade.

Estilo Plánium OS
